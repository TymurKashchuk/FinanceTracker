# FinanceTracker

FinanceTracker — це десктопний застосунок для управління особистими фінансами, побудований на платформі .NET із використанням WPF. Застосунок дозволяє додавати, редагувати та аналізувати доходи, витрати й цілі за допомогою інтуїтивного інтерфейсу та інтерактивних діаграм.

## Основні можливості

- **Додавання та редагування фінансових операцій**  
  Легко створюйте записи про доходи та витрати, організовуйте їх по категоріях.

- **Перегляд фінансової статистики**  
  Візуалізуйте динаміку витрат і доходів за допомогою інтерактивних графіків (LiveCharts).

- **Збереження історії і пошук**  
  Швидко знаходьте та аналізуйте попередні транзакції.

- **Зберігання даних локально**  
  Дані зберігаються у SQLite-базі через Entity Framework Core.

## Технологічний стек

- **.NET 8.0 (WPF)**
- **C#**
- **SQLite**
- **Entity Framework Core**
- **LiveCharts.Wpf** для графіків

```
FinanceTracker/

    ├── AssemblyInfo.cs

    └── Views/               # Окремі сторінки/вікна
```

## Встановлення та запуск

1. **Клонуйте репозиторій:**
    ```bash
    git clone https://github.com/TymurKashchuk/FinanceTracker.git
    cd FinanceTracker
    ```

2. **Відкрийте рішення у Visual Studio 2022 або новішій.**  
   Переконайтеся, що встановлені .NET 8 SDK та розширення для WPF.

3. **Налаштуйте SQLite:**  
   Ніяких додаткових дій не потрібно — база створюється автоматично при першому запуску.

4. **Запустіть проект `FinanceTracker.wpf`:**
   - Оберіть FinanceTracker.wpf як стартовий проект.
   - Натисніть `Start` (F5) для запуску застосунку.
